An SDOF system with stiffness `k`, unit mass `m=1`, and damping coefficient `c` is assumed. Natrual vibration period of the system is `T=0.4 sec` and the damping ratio is `5%`. For seismic protection, an energy dissipating mechanism is added to the system with the attached hysteresis properties.
